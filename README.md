Este repositório reúne algumas **landing pages** desenvolvidas com foco em práticas modernas de **CSS**, priorizando **responsividade**, **SEO** e **acessibilidade**.
Acesse aqui: https://carolinnevictoria.github.io/landing-pages/

---

## Sobre o projeto

O projeto tem como objetivo aplicar, na prática, conceitos fundamentais do desenvolvimento front-end, criando páginas bem estruturadas, visualmente agradáveis e adaptáveis a diferentes dispositivos.

As landing pages foram construídas utilizando **HTML** e **CSS**, com atenção à organização do código e à experiência do usuário.

---

## Objetivos

- Desenvolver layouts modernos e responsivos  
- Aplicar boas práticas de SEO  
- Garantir acessibilidade para diferentes usuários  
- Utilizar HTML semântico corretamente  
- Explorar recursos avançados de CSS  

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Media Queries  

---

## Responsividade

As páginas foram projetadas para se adaptar a diferentes tamanhos de tela:

- 📱 Mobile  
- 📲 Tablet  
- 💻 Desktop  

Utilizando layouts fluidos e **media queries** para melhor experiência em qualquer dispositivo.

---

## SEO (Search Engine Optimization)

Boas práticas aplicadas:

- Estrutura semântica com `header`, `main`, `section`, `footer`  
- Hierarquia correta de títulos (`h1` a `h6`)  
- Organização de conteúdo  
- Melhor leitura por motores de busca  

---

## Acessibilidade

O projeto considera:

- Uso de HTML semântico  
- Imagens com atributo `alt`  
- Estrutura navegável por teclado  
- Contraste adequado de cores  
- Compatibilidade com leitores de tela
